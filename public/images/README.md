# Image placement

Put your thumbnails in these exact paths:

- `public/images/education/university-of-toronto.png`
- `public/images/experience/wheeler-lab.png`
- `public/images/experience/helmholtz-munich.png`

For the site icon + favicon, use the same source artwork and place:

- `public/favicon.ico` (standard favicon)
- `public/favicon.svg` (used by the site header icon and browser tab)
- `public/apple-touch-icon.png` (optional iOS home icon)

The top-left site icon in the page header is wired to `/favicon.svg`, so one icon design is reused in both places.
